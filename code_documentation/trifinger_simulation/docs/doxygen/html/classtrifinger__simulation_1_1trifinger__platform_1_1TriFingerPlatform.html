Initialize.

Parameters
visualizationSet to true to run visualization.
initial_robot_positionInitial robot joint angles
initial_object_poseInitial pose for the manipulation object. Can be any object with attributes position (x, y, z) and orientation (x, y, z, w). This is optional, if not set, a random pose will be sampled.
enable_camerasSet to true to enable rendered images in the camera observations. If false camera observations are still available but the images will not be initialized. By default this is disabled as rendering of images takes a lot of computational power. Therefore the cameras should only be enabled if the images are actually used.
time_step_sSimulation time step duration in seconds.

valid
